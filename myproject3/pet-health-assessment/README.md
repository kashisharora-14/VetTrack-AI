# Pet Health Assessment (React + Tailwind)

## Run

```bash
cd pet-health-assessment
npm install
npm run dev
```

Open `http://localhost:5174`.

## Included

- Step 1: Visual assessment upload with drag-drop, preview, tips
- Step 2: Behavior assessment sliders + observations + dynamic AI context panel
- Step 3: AI processing animation with rotating statuses and staged progress
- Step 4: Diagnostic results with risk score, badge, summary, actions, vets, CTAs

## Notes

- Uses mock data only (no backend calls)
- Existing Flask routes remain unchanged
